ROMAPK API

Intégrez les applications et les roms de Google, Xiaomi et Samsung en toute simplicité.

Introduction

Bienvenue sur ROMAPK API. Notre API vous permet d'accéder aux ROMs et APKs des principaux fabricants de smartphones tels que Xiaomi, Samsung et Google.

Utilisation

Endpoints

  • /xiaomi - Obtenir tous les modèles Xiaomi disponibles
  • /xiaomi/rom/:rom/:model - Obtenir une ROM spécifique pour un modèle Xiaomi
  • /xiaomi/apk/:apk/:model - Obtenir une APK spécifique pour un modèle Xiaomi
  • /samsung - Obtenir tous les modèles Samsung disponibles
  • /samsung/rom/:rom/:model - Obtenir une ROM spécifique pour un modèle Samsung
  • /samsung/apk/:apk/:model - Obtenir une APK spécifique pour un modèle Samsung

Exemples

Voici quelques exemples d'utilisation de notre API:

GET /xiaomi
[
    "RedmiNote13Pro+",
    "Mi10",
    "Mi11"
]

GET /xiaomi/rom/MIUI14/RedmiNote13Pro+
{
    "rom": "MIUI14",
    "model": "RedmiNote13Pro+",
    "version": "14.0.1",
    "downloadLink": "http://example.com/download/rom/miui14/redminote13pro+"
}

GET /samsung/apk/camera/SamsungA20e
{
    "apk": "camera",
    "model": "SamsungA20e",
    "version": "3.4.5",
    "downloadLink": "http://example.com/download/apk/camera/samsunga20e"
}
